# スロット管理 — 実装概要

## 完了時の状態

- スロットを作成・一覧・編集・削除できる
- セッション作成時にスロットを選択すると、テンプレートファイルが worktree に配置される（コピーまたはシンボリックリンク）
- 排他スロットは同時に 1 セッションのみ使用可。使用中は選択不可
- Activity Bar でセッション一覧とスロット管理画面を切り替えられる

## データモデル

### Slot

| フィールド | 型 | 説明 |
| --- | --- | --- |
| id | SlotId (UUID) | 一意識別子 |
| label | String | 表示名 |
| color | String | Mantine カラー名 |
| exclusive | bool | 排他フラグ |
| template_files | Vec\<TemplateFile\> | テンプレートファイル定義 |

### TemplateFile

| フィールド | 型 | 説明 |
| --- | --- | --- |
| relative_path | String | worktree からの相対パス |
| placement | "copy" \| "symlink" | 配置方式 |

### Session（変更）

`slot_id: Option<SlotId>` を追加。

## テンプレート配置

- **copy**: スロットディレクトリ（`~/.tvine/projects/<project-id>/slots/<slot-id>/`）に実体を保存。セッション作成時に worktree へファイルコピー
- **symlink**: 実体は持たない。セッション作成時に `worktree/<path> → リポジトリルート/<path>` のシンボリックリンクを作成
