# ターミナル

## 概要

Claude 端末とは別の、汎用シェル。
各セッションの worktree で git commit、lint、docker compose run/exec 等を実行する場所。

## 表示方式

PTY + xterm.js。ユーザーのデフォルトシェルを起動する。

## 作業ディレクトリ

選択中セッションの worktree ディレクトリ。

## ライフサイクル

- セッション作成時にターミナルの PTY も一緒に作成する
- セッション削除時にターミナルのプロセスを強制終了する
- tvine 再起動時もセッション復元と同時にターミナルを作成する（特別な処理なし）

## セッション切り替え時

セッションごとにターミナルを持つ。切り替えると対応する PTY が表示される。
docker compose up 等の長時間プロセスを残したまま別セッションに切り替えて戻れる。

## 開閉

- トグルで開閉できる
- 閉じた状態ではアイコンバーが残る
- アイコンバーをクリックで再び開く
