# セッション一覧

## 概要

左サイドバーに表示される、全セッション（worktree）の一覧。
各セッションのステータスが一目でわかり、クリックで切り替える。

## 表示項目

各セッションカードに表示する情報：

- ブランチ名
- ステータス（Busy / Waiting / Idle）
- ベースブランチ名
- スロットのラベル
  - スロット設定時は枠の色もスロットに応じて変更

## 操作

| 操作 | 動作 |
| --- | --- |
| クリック | セッション選択。Claude 端末・差分表示が切り替わる |
| ℹ️ ボタン | セッション情報モーダルを開く |
| 削除ボタン | セッション削除モーダルを開く |

## 並び順

作成順（新しいものが下）。

## 空状態

セッションが 0 件のときは「+ New でセッションを作成」のような案内を表示。

## 情報モーダル

ℹ️ ボタンから開く。カードで見切れた情報をフル表示する。

- ブランチ名（フル表示）
- ベースブランチ
- ステータス
- スロットのラベル → クリックでスロット編集モーダルを開く（→ スロット管理.md）
