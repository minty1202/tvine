# 画面構成

## 全体レイアウト

```text
┌──────────┬──────────────────┬──────────┬──┐
│ [+ New]  │  Claude 端末     │ Changes  │⌨ │
│          │  (xterm.js)      │┄┄┄┄┄┄┄┄┄┄│  │
│ ● feat…🟢│                  │ unified  │  │
│ ● fix… 🟡│                  │ diff     │  │
│ ● ref… ⚫│                  │          │  │
│          │                  │          │  │
│          │                  │          │  │
└──────────┴──────────────────┴──────────┴──┘
```

## エリア一覧

| エリア | 位置 | 役割 | 詳細 |
| --- | --- | --- | --- |
| 作成ボタン | 左上 | 新規セッションを作成するボタン | セッション作成.md |
| セッション一覧 | 左 | worktree の一覧とステータス表示 | セッション一覧.md |
| Claude 端末 | 中央 | 選択中セッションの Claude Code 表示 | Claude端末.md |
| 差分表示 | 右 | 選択中セッションの git diff | 差分表示.md |
| ターミナル | 右端 | 汎用シェル | ターミナル.md |

## サイズ

| エリア | 幅 |
| --- | --- |
| セッション一覧 | 15% 固定 |
| Claude 端末 | flex（残り全部） |
| 差分表示 | 20%（トグル。閉じるとアイコンバー） |
| ターミナル | 20%（トグル。閉じるとアイコンバー） |

## リサイズ

ドラッグによるリサイズは MVP では対応しない。

## 画面遷移

メイン画面から開くモーダル：

| トリガー | 遷移先 | 詳細 |
| --- | --- | --- |
| + New ボタン | セッション作成モーダル | セッション作成.md |
| 削除ボタン | セッション削除モーダル | セッション削除.md |
| ℹ️ ボタン | セッション情報モーダル | セッション一覧.md |
| 情報モーダル → スロットラベル | スロット編集モーダル | スロット管理.md |
| 作成モーダル → スロット作成 | スロット作成モーダル | スロット管理.md |

## テーマ

ダークモードのみ（MVP）。ターミナルとの親和性を優先。
